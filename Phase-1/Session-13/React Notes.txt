Installation of React.js
----------------------------
1. DOWNLOAD NODE JS
-> link:https://nodejs.org/en/download

2. CHECK THE VERSION OF NODE USING CMD
-> Open cmd> node -v

3. INSTALL REACT USING NODE
-> npm install -g create-react-app
