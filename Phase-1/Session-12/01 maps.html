Maps in Java Script

A Map Holds Key-value Pair, Where the key can be any datatype

a Map alwways remember the original order of insertion of keys

We can get the size of map any time

    Map has many methods
  1. new Map(): This will create new Map Object
  2. set():sets the value for a key in a map
  3. clear(): Remove all the elements from the map
  4. has(): Returns true if the key exist
  5. get(): get the value for a key in a map
  6. size(): returns the number of Map elements
